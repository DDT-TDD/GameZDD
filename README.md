## 🐛 Troubleshooting

### Game won't start in browser?

**Solution**: Make sure you're opening `launcher.html` with a modern browser (Chrome, Firefox, Edge). Right-click → "Open with" → Choose your browser.

### Save files not working?

**Solution**: Make sure your browser allows localStorage. Check browser settings for "Block third-party cookies" and ensure it's not blocking the game.

### Performance issues?

**Solution**: 
1. Close other browser tabs
2. Update your browser to the latest version
3. Check that hardware acceleration is enabled in browser settings

### Can't hear sound?

**Solution**:
1. Check if sound is muted in game settings (⚙️ button)
2. Check your system volume
3. Try different browsers if issue persists

## 🎓 How to Add More Games

GameZDD is designed to be expandable! To add a new game:

1. **Create a new folder** in `games/` directory
2. **Copy the structure** from an existing game (e.g., `pacman-maze`)
3. **Include the game engine**:
   ```html
   <script src="../../engine/gameEngine.js"></script>
   <script src="../../engine/audioManager.js"></script>
   <script src="../../engine/inputHandler.js"></script>
   <script src="../../engine/saveManager.js"></script>
   ```
4. **Use the SaveManager** for progress tracking:
   ```javascript
   const save = new SaveManager('your-game-name');
   save.save({ currentLevel: 5, totalLevels: 20 });
   ```
5. **Add your game** to the launcher array in `launcher.js`
