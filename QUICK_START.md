## 🎮 Explore All 16 Completed Classic Games:
All 16 classic games in the collection are fully implemented, functional, and visually optimized using themed Orbitron and Rajdhani typography and responsive setups. Select from Puzzle, Maze, Action, and Bubble categories right from your Dashboard!

- **Maze category**: Pac-Man, Maze Munch, Hero Quest, Tunnel Quest, Box Pusher, Sokoban Master.
- **Action category**: Super Jumper, Barrel Climber, Chef Rush, Sailor Quest.
- **Puzzle category**: Block Fall, Memory Cards, Block Blast.
- **Bubble category**: Brick Breaker, Bubble Pop, Marble Chain.

## 🚀 To Build or Launch Desktop App
This project is configured as an Electron-ready application.

### Run in Dev Mode:
```powershell
npm run dev
```

### Build or Pack the Executables:
To build Windows desktop native setups:
```powershell
npm run build:win
```
The packaged installers will be generated under the [dist/](dist/) directory.
