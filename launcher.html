❓ HOW TO PLAY

🎮 Controls

  • Arrow Keys / WASD — Move character
  • Space / W / Up — Jump or shoot
  • ESC — Pause / return to menu
  • Mouse — Click buttons and aim
  • P — Pause toggle (most games)

🏆 Tips

  • Progress is auto-saved as you play
  • Each game has a Speed Slider — start slow!
  • Use the Fullscreen button for the best experience
  • Green bar = Easy · Yellow = Medium · Red = Hard

📂 Categories

  • Maze — Navigation and puzzle-solving games
  • Action — Platformers and dodge games
  • Puzzle — Memory and logic games
  • Bubble — Ball and block matching games
